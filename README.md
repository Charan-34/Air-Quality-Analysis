# Air Quality Analysis Project

## Overview
This project analyzes air quality data to understand pollution levels across cities.

## Features
- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- Data Visualization
- Outlier Detection (IQR)
- Machine Learning (Linear Regression)
- Statistical Testing (T-test & Z-test)

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Results
- Built a model with R² score ~0.68
- Identified pollution patterns across cities
- Statistical tests showed significant differences in AQI
